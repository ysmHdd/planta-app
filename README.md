Planta - Application Flutter pour gérer l'arrosage de vos plantes.
Mini-projet cross platform réalisé par:
- Yasmine Haddad (DSI33)
- Nour Ben Ammar (DSI33)

Fonctionnalités

- Ajouter/supprimer des plantes
- Rappels d'arrosage
- Authentification utilisateur
- Thème sombre/clair

Technologies

- Flutter
- Firebase Auth & Firestore
- BLoC Pattern
- Clean Architecture

Installation

```bash
git clone https://github.com/ysmHdd/planta-app.git
cd planta-app
flutter pub get
flutter run
```
